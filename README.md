# Nature Music Player

一款音乐播放器，音乐来自[网易云音乐][1]。使用[Bomb后端云][2]作为代理服务器。

请使用 Chrome、Firefox、IE11 浏览。

## 功能特性

    1. 音乐搜索（歌曲名、歌手、专辑）
    2. 音量调节（方向键 &uarr; &darr;）
    3. 进度调节（方向键 &larr; &rarr;）
    4. 歌词显示
    5. 循环播放
